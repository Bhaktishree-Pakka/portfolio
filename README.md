# portfolio
A fully responsive developer portfolio built using Next.js and React, designed to showcase my projects, skills, and experience with a modern UI, smooth navigation, and optimized performance.

---

## 🚀 Live Demo

👉https://portfolio-nu-lac-27.vercel.app/

---

## 📌 Features

- ⚡ Fully responsive design (mobile + desktop)
- 🎨 Clean and modern UI
- 🌙 Light/Dark mode support
- 🧩 Projects showcase section
- 📱 Smooth navigation between sections
- ⚡ Fast performance with Next.js

---

## 🛠️ Tech Stack

- Next.js
- React
- Javascript
- CSS

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/Bhaktishree-Pakka/portfolio.git
```

### 2. Navigate to the project folder

```
cd portfolio
```

### 3. Install dependencies

```
npm install
```

### 4. Run the development server

```
npm run dev

```
